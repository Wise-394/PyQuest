[b]List[/b]
[i]One variable. Endless possibilities.[/i]

[hr]

A [b]list[/b] is an ordered, mutable collection of items in Python. Unlike a single variable that holds one value, a list can hold [b]multiple values[/b] under a single name — making it one of the most versatile data structures in Python.

Lists can store items of any data type, and their contents can be modified after creation.

[b]In this level, you will:[/b]
[ul]
Create and initialize lists using square brackets [b][][/b]
Access individual items using their [b]index[/b]
Modify, add, and remove items from a list
[/ul]
