Python supports several built-in [b]data types[/b] that define the kind of value a variable holds. Choosing the correct data type is essential for writing accurate and efficient programs.

The four fundamental data types in Python are [b]int[/b] (whole numbers), [b]float[/b] (decimal numbers), [b]str[/b] (text), and [b]bool[/b] (True or False).
