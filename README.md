## Game Concept

- **Jump, run, and explore** colorful 2D levels.  
- **Solve coding challenges** to unlock doors, chests, collect items, and progress.  
- **Learn Python basics and concepts** in an interactive way.  
- **Track your coding progress** as you play.

## Planned Features

- Multiple platforming levels with increasing difficulty.  
- Integrated Python code challenges and puzzles.  
- Rewards and achievements to motivate learning.
